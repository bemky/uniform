Grid

Similar to Bootstrap 4 with some modifications. Uses flexbox with a host of helper options.

Breakpoints

Any helper can have -sm, -md, -lg, or -xl added to cause helper to only go into effect for that breakpoint.

xs = 480px

sm = 768px

md = 992px

lg = 1200px
