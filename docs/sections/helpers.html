Helpers

Helper classes are very basic rules that modify usually apply 1 or 2 attributes to an element. They are meant to make UI process fast and easy.

Modifier Hierarchy

.attribute[-direction][-size][-scope] 
@media max-width: 719px){
    .pad-top-2x-sm{
        padding-top: 2em;
    }
}

Attributes

These attributes follow the same structure for the given attribute

pad padding
margin margin
text font-size

Direction

-left left
-right right
-top top
-bottom bottom
-h left and right
-v top and bottom

Size

-quarter 0.25em
-half 0.5em
-smaller 0.8em
-small 0.9em
-base 1em
-large 1.2em
-larger 1.4em
-2x 2em
-4x 4em

Scopes

-sm < 720px
-md 720px - 1080px
-lg 1080px - 1440px
-xl 1440px +
