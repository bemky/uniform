Thumb

The size of the image doesn't matter, it will fill the shape of the div.uniformThumb using object-fit: cover;.

Requires height of uniformThumb to be set or use responsive aspect ratio liner.

Aspect Ratio Liner

<div class="uniformThumb">
    <!-- Aspect Ratio Liner -->
    <div class="aspect-4x3"> 
        <img src="https://unsplash.it/200/200/?random" width="200" height="200">
    </div>
    <!-- END Aspect Ratio Liner -->
</div>
