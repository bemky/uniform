UniformSelect

Turn any select box into a more advanced full featured dropdown UI element, while maintaining the html simplicity of a basic select element.

Initialize

$(document).ready(function(){
    $('.uniformSelect').uniformSelect(options);
})

Becomes

<select class='uniformSelect'>
    <option>An Option</option>
</select>
multiple 
<select class="uniformSelect" multiple>
    <option>An Option</option>
</select>

Limits dropdown to 8 options (optional)

<select class="uniformSelect">
    <option>Option 1</option>
    <option>Option 2</option>
    <option>Option 3</option>
    <option>Option 4</option>
    ...
    <option>Option 19</option>
</select>

Responsive

On mobile dropdown of options becomes full screen modal of options.

Options

$(document).ready(function(){
    $('.uniformSelect').uniformSelect({
        limit: 8,
        class: 'uniformButton green',
        showAll: function(select_options){...}
    });
})
Option Accepts Description Default
showAll function(select_options) Function run called when "Show All is clicked" Removes "Show All" and reveals all hidden options
limit int, false Limit number of visilbe options. Set to false to not limit. 8
class string Classes to append to uniformSelect-edit (main button) ''
