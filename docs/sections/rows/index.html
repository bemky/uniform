Rows

Rows are great for table like data that needs to transform into cards to be responsive.

uniformThumb

Uses uniformThumb to use any size image to fill the space of the thumbnail.

uniformRow-body

Use uniformRow-body for content of row. All child elements are treated as columns 

<a class="uniformRow" href="#rows">
    <span class="uniformThumb">
        <img src="https://unsplash.it/400/300/?random">
    </span>
    <div class="uniformRow-body">
        <span>
            John Doe
            <span class="text-subtle block">ABC Designs</span>
        </span>
        <span>
            Garland, TX
        </span>
        <span>
            Texas Ranger Fan
        </span>
    </div>
</a>
