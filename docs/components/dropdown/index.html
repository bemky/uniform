Dropdown JS

Toggle a dropdown on an anchor. 

import {Dropdown} from 'uniform';
new Dropdown({
    anchor: document.getElementById('example-1'),
    content: "<div class='uniformPointer shadow shadow-opacity-40 bg-white pad'>Here's Content</div>"
})

Options

Option Type Description Default
anchor Element Element to anchor content to. REQUIRED
content HTML HTML to render into popover. REQUIRED
align String How to align the popover to the anchor left|right|center|#px top|center|bottom|#px 'center bottom' OPTIONAL
container Element Element to append popover to. document OPTIONAL
