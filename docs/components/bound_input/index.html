BoundInput JS

Renders an input, select, button, or textarea that is bound to the designated object. Changing input updates object attribute, and changing object's attribute updates the input.

Text Input
object.foo: 
new BoundInput({
  record: record,
  attribute: "foo",
  type: "text"
}).el
