Modal JS

Render any content into a modal 

import {Modal} from 'uniform';
new Modal({
    content: '<div class="bg-green-20 border border-green text-green-80 pad rounded">Example Content</div>'
}).render()

Options

Option Type Description
content String or Element HTML or Element to render
class, name, id, value String Accepts any html attribute that is then rendered on the modal element.

Events

Event Description Arguments
rendered After modal is rendered none
closed After modal is closed none
