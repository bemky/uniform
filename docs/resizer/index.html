UniformResizer

Responsive layouts are great when the window is the container you care about. UniformResizer brings responsive helpers to elements that window gets. Initialize any element and it will toggle size classes: sm-size, md-size, lg-size, and xl-size. All Uniform classes that can be scoped to size also adhere to these classes. 

// javascript
new Uniform.Resizer({
    el: document.querySelectorAll("#target")
})

If an object is 1200px wide, it will have both md-size and lg-size, because it is both larget than 720px and 1080px.

The small class, sm-size is treated differently. It can only have sm-size if it is under 720px.

The media queries for Uniform are setup the same exact way with min-width for sm and max-width for the other sizes
