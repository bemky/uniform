Programmatically

new UniformModal({
content: "This is your modal content",
}).render()
Options Accepts Description
content string | $(element) | function Rendered as content, can be a function
klass string HTML class to append to modal container
Events Description
rendered After modal is rendered and shown
closed After modal is closed before being removed
Methods Description Example
on(event, handler) Add event listener modal.on('closed', function(modal_instance){...})
close Close modal modal.close()

Using JQuery

<button class="launchUniformModal" data-modal-target="#modal-content">
Launch Modal
</button>
$('.launchUniformModal').click(function(){
$(this).uniformModal();
});
Data Options Description
data-modal-content String of html to render in modal
data-modal-target Instead of providing data-modal-content you can provide a jquery selector of target element. That will be used to render in modal.
data-modal-klass HTML class to append to modal container

Events

Events fired on the launchUniformModal element.

Events Description
modal-rendered After modal is rendered and shown
modal-closed After modal is closed before being removed
