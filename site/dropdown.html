Programmatically

new UniformDropdown({
    el: $(this),
    content: "This is your modal content",
}).render()
Options Accepts Description Default
content string | $(element) Rendered as content
align left, center, right Align dropdown to trigger element center
trigger click, focus, mouseover... Trigger to show dropdown. Space separated. click focus
show_arrow true | false Toggle showing arrow/triangle outside of dropdown true
hide_sm true | false Toggle showing dropdown in 'sm' browsers (< 720px) false
Events Description
shown After dropdown is shown
hidden After dropdown is hidden
Methods Description Example
on(event, handler) Add event listener dropdown.on('hidden', function(modal_instance){...})
show Close modal dropdown.close()
hide Hide dropdown dropdown.close()
